# Panduan berkontribusi
Terdapat tiga jalan bagi Anda jika berkontribusi melalui Github:
1. Memberi bintang :star2: pada repository ini.
1. Mengirimkan [issue](https://github.com/kalabali/penyalin-wariga/issues).
1. Jangan mengirimkan pull request tanpa ada issue agar tidak overlapping
1. Mengirimkan pull request ke branch `master`.
